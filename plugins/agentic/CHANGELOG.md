## 1.1.0 — 2026-06-24

- Clarified that the routing targets (`claude-api`, `agent-sdk-dev`, `mcp-server-dev`, `plugin-dev`) are official Anthropic plugins not shipped in this marketplace — add them from their own source before they resolve.
- Refreshed default model ids in the README and `agentic-app-setup` skill to the current latest (Opus 4.8 / Sonnet 4.6 / Haiku 4.5), reframed as "latest — check `claude-api`" rather than pinned values.
- Manifest enriched with `$schema`, `homepage`, `repository`, `license`, `keywords`, and category metadata.
